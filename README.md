# Landing Page Starbucks

Esse projeto foi desenvolvido a partir das aulas do curso de html e css da B7Web com o objetivo de aprimorar as habilidades nessas stacks e aplicar ao projeto boas práticas de responsividade com o objetivo de melhorar a experiência do usuário.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Instalação e Uso

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd nome-do-projeto
   ```
3. Abra o arquivo `index.html` no navegador.

## Como Contribuir

1. Faça um fork do projeto.
2. Crie uma nova branch: `git checkout -b minha-feature`
3. Faça suas alterações e commit: `git commit -m 'Adicionando uma nova funcionalidade'`
4. Envie para o repositório remoto: `git push origin minha-feature`
5. Abra um Pull Request.
